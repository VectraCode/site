Control+

Privacy Policy

This policy describes the current Control+ app behavior. Control+ is a habit tracker designed to work without an account.

Last updated: July 7, 2026 Brand: VectraCode Publisher / legal contact: LEGAL_PUBLISHER_CONTACT Contact: support@vectracode.net Public policy URL: https://vectracode.net/control/privacy

Overview

Control+ is built for local habit tracking. The current version stores habit data on the user's device and does not include cloud sync, advertising, or in-app analytics.

Data Stored On The Device

Control+ stores the following data locally:

  • habit names;
  • habit frequency and custom interval settings;
  • habit color and icon choices;
  • habit creation dates;
  • completed habit dates;
  • reminder settings, such as enabled state and reminder time;
  • app settings, such as theme preference.

This data is stored locally by the app so the user can track habits and see progress over time.

Backups, Import, And Export

When the user exports data, Control+ creates a backup file on the device and uses the system share or save flow. The user chooses where that file is saved or sent.

When the user imports data, Control+ reads only the backup file selected by the user. Imported habits are added to the existing list and do not replace current habits unless the user separately deletes data.

Notifications

If the user enables reminders, Control+ schedules local notifications on the device. Reminder scheduling is used only to show habit reminders. On Android, the app may ask for notification permission and may use system startup events to keep scheduled reminders available after the device restarts.

Data Sharing

Control+ does not sell personal data, does not show ads, and does not send habit data to a Control+ server.

Habit data may leave the device only when the user chooses to export or share a backup file through the system share or save flow.

Purchases

Control+ Pro is planned as a one-time purchase. Store purchase processing may be handled by Apple, Google, or another platform provider according to that provider's terms and privacy practices. Public release builds should use the store-backed purchase and restore flow.

Data Deletion

The app includes an option to delete all local habit data, completion history, and local reminder settings from the device. Users can also remove the app from the device to delete app data according to the operating system behavior.

Children's Privacy

Control+ is not designed to collect personal information from children. The current version does not include accounts, social features, advertising, or analytics.

Changes

This policy may be updated when Control+ adds new features, such as real purchases, sync, analytics, or account-based functionality. Material changes should be reflected in this document before release.
